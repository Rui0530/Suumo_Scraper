# Suumo_Estate_Scraper

SUUMO（不動産情報サイト）から物件情報を自動収集し、CSV形式で保存するオブジェクト指向型スクレイパーです。

## ✨ 特徴
- **クラス設計（Object-Oriented）**: 処理を `Estate_Scraper` クラスにまとめ、初期設定・アクセス・データ取得・保存を独立させているため、メンテナンス性に優れています。
- **動的スクロール実装**: 「次へ」ボタンが画面外にある場合を考慮し、JavaScriptによる `scrollIntoView` を実行してからクリックする堅牢なページ遷移ロジックを搭載。
- **逐次保存（Append Mode）**: 全件取得後に保存するのではなく、1ページごとにCSVへ追記（append）するため、万が一の通信エラー時もそれまでのデータを失いません。
- **ヘッドレスモード対応**: 実行時に `headless=True/False` を切り替えるだけで、ブラウザ表示の有無を選択可能です。

## 🛠 技術スタック
- Python 3.x
- Selenium (Webブラウザ自動操作)
- Webdriver Manager (ドライバ自動更新)
- CSS Selector / Link Text による要素抽出

## 📊 取得項目
以下の4項目を抽出します。
1. 物件名
2. 家賃（管理費等含む）
3. 間取り
4. 住所

## 🚀 実績
- 沖縄県那覇市の物件情報を対象に、**全1,918件**のデータを安定して取得・保存完了。

## 📝 工夫した点
- `f.tell() == 0`: CSV作成時、ファイルが空の時だけヘッダーを書き込む処理を入れ、追記時にヘッダーが重複しないよう制御しています。
- `implicitly_wait(5)`: 要素が見つかるまで最大5秒待機する設定により、読み込み速度の変動に強い設計にしています。
